# Welcome!

This is a document for housing lecture notes I prepared for teaching Combinatorics. The course was designed to be an introduction to combinatorics as a whole targeted toward mathematicians. 

We covered foundational combinatorics materials and dove into some more specific topics as we progressed. The 15-week course felt like it passed quickly, but we covered a lot of material that is fun to look back on.

There are warmups at the beginning of each section (just like in-class) and plenty of exercises at the end of each section if you would like a challenge!
